---

Slack integration is a Pro feature and lives in the [`lynox-pro`](https://github.com/lynox-ai/lynox-pro) repository.

The Pro Slack bot connects to LYNOX's MCP server via Socket Mode and supports multi-message thread streaming, interactive approval buttons, follow-up suggestions, voice message transcription, and file handling.

See the [Extension Points](/extension-points/) documentation for how Pro features integrate with the core via Mode Registry, Orchestrator Hooks, and CLI Command Registry.
